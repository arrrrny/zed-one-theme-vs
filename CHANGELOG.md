# Change Log

All notable changes to the "zed-one-theme" extension will be documented in this file.

## [0.3.0] - 2026-01-29

### Added
- Comprehensive UI borders for Sidebar, Tabs, Activity Bar, and Panels to accurately reflect ZED's structured layout.
- Full 16-color ANSI terminal palette (Normal and Bright) from the official ZED source.
- Specific `tab.unfocusedActiveBorder` support to maintain visual context across editor panes.
- New `focusBorder` mapping for inputs to match ZED's focused tokens.

### Fixed
- Terminal background color shifted from generic dark to ZED's specific surface color.
- Message box and system notification backgrounds aligned with the primary UI theme.
- Fixed lint warnings by removing non-standard theme properties.

## [0.2.0] - 2026-01-29

- Refactored theme color tokens for better consistency between Dark and Light variants.
- Improved syntax highlighting for specialized language scopes.

## [0.1.0] - 2026-01-28

- Initial release.
